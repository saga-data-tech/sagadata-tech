# Sagadata.tech

Official website for Saga Data LLP.

## Overview

A high-performance static website designed with a dual-theme identity:

- **Warm & Academic**: "The Journal" vibe 
- **Cool & Industrial**: "The Grid" vibe 

## Architecture

- **Stack**: HTML5, Vanilla CSS, Vanilla JS.
- **Hosting**: GitHub Pages.
- **Deployment**: Automatic via GitHub Actions (`.github/workflows/deploy-pages.yml`).

## Development

- Open `index.html` in any browser to view locally.
- Styles are in `assets/css/style.css`.
- Theme logic is in `assets/js/main.js`.
